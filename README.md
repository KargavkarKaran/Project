<h3>Project Overview</h3>
<p>
            An IoT-based Smart Agriculture System designed to improve crop productivity,
            optimize resource usage, and ensure transparent agricultural supply chains.
            The system integrates IoT sensors, AI-driven analytics, and Blockchain
            technology to enable precision farming, automated irrigation, and tamper-proof
            supply chain tracking.
        </p>
        <p>
            Built with a scalable, modular, and real-time architecture, this system is
            suitable for both small-scale farms and large agricultural deployments.
        </p>

<h3>Key Features</h3>

<h5>Real-Time Farm Monitoring</h5>
        <ul>
            <li>Soil moisture monitoring</li>
            <li>Temperature and humidity tracking</li>
            <li>Water level and rainfall detection</li>
            <li>Pest activity monitoring</li>
        </ul>

<h3>AI-Based Smart Irrigation</h3>
        <ul>
            <li>Automated irrigation based on soil and crop requirements</li>
        </ul>

<h5>AI & Machine Learning</h5>
        <ul>
            <li>Crop health monitoring</li>
            <li>Leaf disease detection using deep learning</li>
            <li>Predictive analytics for weather and pest outbreaks</li>
        </ul>
